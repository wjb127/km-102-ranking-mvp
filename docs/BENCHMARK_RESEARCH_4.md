# 벤치마킹 리서치 #4 — 23 사이트 종합

원문 보관용. 이전 3종 리서치 보완 + 신규 사이트 8곳 발굴 (Letterboxd, RYM, IMDb, Theqoo, Naver Cafe, BoxRec, BJJ Heroes, FotMob 등).

## 1. 벤치마킹 사이트 23개

### A. 직접 경쟁 (MMA 전문)

#### 1) Sherdog — sherdog.com
- 성격: 영문권 MMA 1세대 데이터 + 뉴스 + 포럼
- 겹침: 선수 DB(Fight Finder), 일정/결과, 랭킹, 포럼
- 배울 점: Fight Finder 깊이 — 군소단체까지 누적, 전적 6필드 표준 (상대/날짜/방식/라운드/시간/이벤트)
- 따라하면 안 되는 점: UI 2000년대급, 광고 난잡, 포럼 분리 도메인 → 분리하지 마라
- 수익화: 디스플레이 + 베팅 odds 제휴

#### 2) Tapology — tapology.com
- 성격: 데이터 + 픽엠 게임화 커뮤니티
- 겹침: 프로필, 일정, 랭킹, 픽엠/예측, 포럼, 사용자 등급(벨트)
- 배울 점: 게임화 — 픽엠 정답 50점, 방식/라운드 적중 보너스, 흰띠→검은띠 승급. 재방문 핵심
- 따라하면 안 되는 점: 정보 밀도 과다 → 모바일 퍼스트로 단순화
- 수익화: 픽엠 스폰서십, 베팅 affiliate

#### 3) MMA Fighting (SB Nation) — mmafighting.com
- 성격: 뉴스 + 팟캐스트 + 영상
- 배울 점: 장문 + 팟캐스트 임베드 + 댓글 트리오, 정기 시리즈("The MMA Hour")로 재방문
- 따라하면 안 되는 점: SBN 템플릿 차별화 약함

#### 4) MMA Junkie — mmajunkie.usatoday.com
- 배울 점: 매일 발행 정기 코너("Twitter Mailbag", "Trading Shots")
- 따라하면 안 되는 점: USA Today 흡수 후 커뮤니티 약화

#### 5) ESPN MMA — espn.com/mma
- 배울 점: Fightcenter UX — 메인/프렐림 토글, 라이브 시 라운드별 점수 우측 실시간 갱신. 라이브 경기 페이지 표준
- 따라하면 안 되는 점: 12개월 활동 컷 랭킹 → 한국 팬덤은 은퇴/공백 선수도 보고 싶어함

#### 6) UFC.com
- 배울 점: 선수 상세 시각화 — Significant Strikes by Position 도넛 차트, Striking Accuracy 게이지. 인포그래픽화
- 따라하면 안 되는 점: UFC 선수만, 단체 마케팅 톤

#### 7) MMA Korea / MK Sports MMA
- 배울 점: 공백 시장 — 한국어로 DB+커뮤니티+랭킹 통합된 곳 부재. 우리가 1번 가능
- 따라하면 안 되는 점: 단발성 기사뿐, 누적 자산 없음

### B. 부분 기능 유사

#### 8) BoxRec — boxrec.com
- 배울 점: 알고리즘 자동 랭킹 + 위키 통합. 자원봉사 에디터
- 따라하면 안 되는 점: UI는 옛날 — 데이터 구조만 참고

#### 9) FightMatrix — fightmatrix.com
- 배울 점: CIRRS/Elo 랭킹, 역대 랭킹 타임라인, 부상/공백 패널티 계산. "왜 이 순위인가"를 데이터로 설득
- 따라하면 안 되는 점: 통계 마니아용 → 알고리즘 백엔드, UI는 단순

#### 10) BJJ Heroes — bjjheroes.com
- 배울 점: 계보(Lineage) 트리 — 사제관계 사슬. 한국 선수에 적용 시 콘텐츠
- 따라하면 안 되는 점: 글 길고 정적, SEO엔 좋지만 재방문 약함

### C. 다른 스포츠 구조

#### 11) Transfermarkt — transfermarkt.com
- 배울 점: 시장가치 투표 — 선수마다 "올라야/내려야/유지" 1클릭(비로그인 가능), 트렌드 그래프. judge 권한제 — 활동 많은 유저에게 편집 권한 위임(어드민 부담↓)
- 따라하면 안 되는 점: 시장가치 컨셉 자체는 MMA 부적합

#### 12) Basketball-Reference — basketball-reference.com
- 배울 점: 선수 페이지 좌측 인덱스 네비 — Per Game/Totals/Advanced/Splits 탭. MMA = "전적/라운드별/타격위치별/상대 랭킹별/미디어" 응용
- 따라하면 안 되는 점: 데스크탑 표 위주 → 모바일 가로스크롤 유혹 X

#### 13) FotMob — fotmob.com
- 배울 점: 라이브 매치 페이지에 reactions/polls 통합. "이 라운드 누가 잡았어?" 라이브 투표 → 체류 폭발
- 따라하면 안 되는 점: 커뮤니티 약함 → 라이브+게시판 연결

#### 14) ESPN (전체)
- 배울 점: 종목 간 통일된 IA — 홈/스코어보드/일정/팀/선수/통계 5~7 메뉴 표준

### D. 데이터 UX 강한

#### 15) Wikipedia
- 배울 점: 인포박스 표준 — 사진+닉네임+신체+스탠스+팀+데뷔+전적 한 표. 본문은 경력/스타일/사생활 섹션
- 따라하면 안 되는 점: 위키 편집 권한 개방 → 검증 에디터만

#### 16) IMDb
- 배울 점: Watchlist + 별점 + 트리비아 + 명대사. MMA 적용: 관심 선수/경기 와치리스트, 명언, 트리비아(체중감량 일화), 가중평균 별점

#### 17) Letterboxd
- 배울 점: 다이어리 형식 — 본 경기 기록, 0.5~5.0 별점, 짧은 리뷰, 친구 활동 피드, 좋아하는 4선수 핀. 소셜 디스커버리 — 비슷한 취향 팔로우. MMA 팬덤 톤에 가장 적합
- 따라하면 안 되는 점: 데이터 깊이 약함 → 데이터+Letterboxd 톤 결합

#### 18) RateYourMusic
- 배울 점: 가중치 차트 알고리즘(어뷰징 방지), 유저 리스트 차트 인기. MMA = "역대 KO 25선", "내 인생 경기 50선" → SEO+재방문
- 따라하면 안 되는 점: UI 1990년대급

### E. 커뮤니티 UX 강한

#### 19) Reddit r/MMA
- 배울 점: 이벤트 라이브 메가스레드(자동 생성, 댓글 수만개), Flair(응원 선수 플래그), AMA
- 따라하면 안 되는 점: 익명성 양극화 → 소프트 실명/등급으로 톤 관리

#### 20) DCInside UFC 종합격투기 마이너 갤러리
- 배울 점: 한국 MMA 팬덤 본진 — 속보성, 고닉 vs 유동닉, 짤방 문화. 모니터링 필수
- 따라하면 안 되는 점: 매니저 검열 → 운영 정책 명문화 + 신고/제재 로그 투명화

#### 21) FMKorea 격투기 게시판
- 배울 점: 포텐(추천 누적 메인 노출), 닉네임 기반 익명성
- 따라하면 안 되는 점: 정치/혐오 톤 → 카테고리 분리 + 강제 신고 임계점

#### 22) Theqoo
- 배울 점: '무명의 더쿠' 익명화 + 한 글 안에서만 숫자 식별(원덬1, 원덬2). 닉네임 트래킹 부담 없이 토론
- 따라하면 안 되는 점: 비가입 글쓰기는 어뷰징 → 가입 필수, 닉만 익명 옵션

#### 23) Naver Cafe 이종격투기 / 앤디훅
- 배울 점: 카페 등급제 + 출석 + 정회원 승급 — 진입장벽 = 충성도. 정회원 전용 게시판으로 락인
- 따라하면 안 되는 점: 등급 너무 빡빡 → 1주차 정회원 가능한 완만한 곡선

## 3. MVP vs 고도화 단계 구분

### 1차 (런칭 필수)
- 선수 DB: 한국 + UFC/PFL/Bellator/ONE/RIZIN 상위 200명
- 선수 상세: 인포박스 + 전적 6필드 표
- 경기 일정: 30일 캘린더 + 메인 카드
- 경기 상세: 카드/프렐림 + 시작시간(UTC+KST) + 중계처
- 게시판: 자유/뉴스/예측/하이라이트 4개 + 추천/비추천 + 댓글
- 프로필: 닉+응원 선수+작성 이력
- GOAT/인기투표: Transfermarkt식 1클릭 ↑/↓ + 트렌드
- 쪽지: 1:1 단순
- 관리자: 신고 처리, 데이터 편집, 공지

### 2차 (+2~4개월)
- 픽엠 게임 (승자+방식+라운드 예측, 리더보드)
- 사용자 등급/벨트 (흰띠→검은띠)
- 라이브 이벤트 페이지 (라운드별 투표+메가스레드)
- 사용자 리스트 ("내 인생 경기 25선")
- 와치리스트 + 알림
- 다이어리(별점+한줄 리뷰)
- AMA 기능

### 3차 (+5~9개월)
- 알고리즘 랭킹 (Elo)
- 계보 트리
- 에디터 권한제
- 팟캐스트/영상 임베드
- 베팅 odds 제휴 / Pro 구독
- 다국어(영어)

## 4. "전문 커뮤니티 느낌" 핵심 10

1. 데이터 깊이 — 군소단체까지 누적
2. 표기 표준화 — 6필드 고정, 약어 일관(UD/SD/KO/TKO/SUB)
3. 출처 명시 — 데이터 행마다 출처+갱신일
4. 약어/용어 hover tooltip
5. 시각적 절제 — 광고 도배 X
6. 라이브 라운드별 점수판
7. 신뢰 가는 랭킹 근거 노출
8. 한국어 전용 깊이 — 인터뷰/팀 변경/체급 변동
9. 추천+비추천 의견 다양성
10. 운영 투명성 — 신고/제재 로그 페이지

## 5. 재방문 장치 10

1. 픽엠 — 마감 카운트다운 + 결과 알림
2. 와치리스트 알림 — 1주/1일/1시간 전 푸시
3. 라이브 이벤트 페이지 — 30분 전 자동 활성
4. 일일 랭킹 변동 — "어제 ↑3 / ↓2" + 사유
5. 활동 점수/벨트 시각화
6. 출석 체크 — 7일 연속 보너스
7. 이 주의 베스트 리스트 메인 노출
8. 댓글 알림 — 답글/멘션
9. 응원 선수 피드 — 해당 선수 글/뉴스만
10. 주간 정기 코너 — 월(주말리뷰)/수(프리뷰)/금(AMA)

## 6. 액션 아이템 체크리스트

### 데이터 모델
- 선수: id, 한글명, 영문명, 닉네임, 생년월일, 키/체중/리치, 스탠스, 팀, 데뷔일, 활동상태, 위키링크, 마지막갱신일
- 전적: 상대선수id, 날짜, 결과(W/L/D/NC), 방식(KO/TKO/SUB/UD/SD/MD), 라운드, 시간, 이벤트id, 단체
- 이벤트: 단체, 이름, 날짜(UTC+KST), 장소, 메인/프렐림, 중계처
- 투표: target_type(선수/경기/GOAT), target_id, user_id, 값(↑/↓), ts
- 게시글: 카테고리, 제목, 본문, 태그(선수), 추천/비추, 신고

### UI
- 모바일 퍼스트, 가로스크롤 금지
- 약어 hover tooltip 컴포넌트 통일
- 선수 인포박스 컴포넌트 (Wikipedia식)
- 전적 표 컴포넌트 (Sherdog 6필드, 상대 클릭→상대 페이지)
- 라운드 점수 위젯 (ESPN식 라이브)
- 1클릭 투표 위젯 (Transfermarkt식, 비로그인 read-only)

### 운영
- 신고 5건 누적 자동 가림 + 검토 큐
- 운영 정책/제재 로그 공개 페이지
- 활동 점수 X 이상 → 데이터 편집 권한
- 모든 데이터 행 출처 1개 이상

### 콘텐츠
- 런칭 시 한국 선수 50 + UFC P4P 15 + 단체별 50 = 최소 150명 사전 입력
- 30일 이벤트 사전 입력
- 정기 코너 3개 요일 픽스
- AMA 후보 명단 사전 컨택

### SEO
- 선수 URL 슬러그 = 영문+한글 (예: `/fighters/jung-chan-sung-coreanozombi`)
- sitemap.ts 자동 (선수/이벤트 동적)
- OG 이미지 1200×630 자동 생성
- JSON-LD Person/SportsEvent

### 수익화
- 광고 슬롯 위치 사전 정의 (게시판 5번째, 상세 본문 중간 1개)
- 베팅 odds 제휴 — 지역 규제 확인 후
- 장기 Pro 구독(광고 제거 + 고급 스탯/리스트 무제한)
