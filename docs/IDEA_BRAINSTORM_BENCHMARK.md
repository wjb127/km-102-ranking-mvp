# 아이디어 브레인스토밍 — 벤치마킹 3종 종합

세 개의 벤치마킹 리서치(18 사이트 × 18 사이트 × 15 사이트)를 클라이언트 와우 모먼트 관점에서 압축·정리.

---

## 0. 한 줄 컨셉

> **"데이터(DB)는 땔감, 커뮤니티는 엔진. 한국 MMA 팬이 '여기 진짜배기다'라고 인정하게 만든다."**

- 데이터 페이지가 곧 커뮤니티 진입점 (Tapology + HLTV)
- 경기 전 → 경기 중 → 경기 후 감정 사이클 전체 커버 (Verdict)
- 한국 통용 표기 + 모바일 우선 + 다크 무드 (FMKorea + DC + Sofascore)

---

## 1. 세 리서치 공통 합의점

세 리서치가 한목소리로 짚은 결론:

| 항목 | 합의 |
|------|------|
| **원형 조합** | Tapology + Sherdog + Verdict + Reddit r/MMA |
| **MVP 핵심 순환** | 경기일정 → 선수상세 → 투표 → 댓글 → 게시판 (5각 순환) |
| **재방문 1순위** | 승부예측(Pick'em) — 경기 전 참여 + 경기 후 정답률 확인 |
| **커뮤니티 1순위** | 자유게시판 X → 경기/선수 상세 댓글부터 시작 |
| **데이터 깊이** | 처음부터 위키급 X → 핵심 카드만, 통계는 2차 |
| **모바일 우선** | 하단탭 + 1터치 투표 + 댓글 하단 고정 |
| **운영 부담** | 위키식 데이터 스카우트 X → 1인 관리 가능 수준으로 시작 |

## 2. 와우 모먼트 후보 (임팩트 × 난이도 매트릭스)

데모할 때 클라이언트가 "오"하게 만드는 것 위주로.

### A. 즉각 와우 (구현 쉬움 × 임팩트 큼) — **데모 1호 후보**

#### A-1. **선수 이름 자동 태그 + 호버 미니카드**
- 게시글/댓글에 `@맥그리거` 입력 → 하이퍼링크 + 호버 시 전적/체급/사진 툴팁
- 출처: HLTV의 핵심 마법
- 효과: 댓글 읽다가 자연스럽게 선수 DB로 빨려 들어감 → 체류시간 폭발

#### A-2. **Tale of the Tape (양 선수 등 맞댐 비교뷰)**
- 경기 상세 진입 시 좌우 선수 사진 + 가운데 키/리치/체중/전적/리치 차트
- 출처: Tapology / ESPN Fightcenter
- 효과: "오 진짜 격투기 사이트네" 즉시 인식. 시각적 임팩트 최대.

#### A-3. **승/패/무 색상칩 + KO/SUB/DEC 약어 칩**
- 전적표가 초록/빨강/회색 + 피니시 약어 자동 색칠
- 출처: Sherdog 표준
- 효과: 표 하나로 "전문성" 시그널. 한 줄로 커리어가 읽힘.

#### A-4. **다크 모드 디폴트**
- 격투기 사진의 피·땀이 살아나는 무드
- 출처: HLTV / OP.GG
- 효과: 진입 0.5초 만에 "전문 사이트" 인식

#### A-5. **한국 통용 표기 (이미 적용됨)**
- 프라치스, 정찬성, 코리안 좀비, 하빕, 마카체프 등 183명 + 124이벤트 + 35베뉴
- 효과: 클라이언트 피드백 직접 반영 → "내 말 들었네" 신뢰감

---

### B. 중간 와우 (1주 이내 구현 가능 × 임팩트 큼) — **데모 2호 후보**

#### B-1. **경기 카드 1탭 승자 예측 → 즉시 % 공개**
- 홈/일정/경기상세 어디서든 1터치, 투표 직후 전체 % 공개
- 출처: Verdict + Tapology
- 효과: 경기 전 참여 허들 0. "어, 나도 한번 찍어볼까" → 등록까지 가는 결정적 한 방.

#### B-2. **GOAT 인기투표 — Up/Down 라이브 랭킹**
- 선수 상세에서 ▲/▼ 1탭, 실시간 순위 변동
- 출처: Reddit + Tapology
- 효과: 클라이언트가 "정찬성" 검색 → ▲ 누름 → 한국 P4P 랭킹 즉시 변동 보임 → 강력한 와우.

#### B-3. **판정 논란 자동 투표**
- 결과 등록 시 분할판정/만장일치 표시 → 분할이면 "너는 누구 승?" 자동 투표 생성
- 출처: MMA Decisions
- 효과: 경기 후 분노형 유저 흡수. 댓글 폭발 트리거.

#### B-4. **이번 주 빅매치 카운트다운 카드**
- 홈 최상단 "UFC 322 D-3" + Tale of the Tape + 예측 % 미니카드
- 출처: UFC.com 이벤트 + Sofascore 캘린더
- 효과: 첫 진입 1초 만에 "이번 주 뭘 봐야 할지" 체감.

#### B-5. **내 픽 정답률 / 뱃지 (화이트~블랙벨트)**
- 프로필에 "이번 달 8승 3패", 누적 적중률, MMA 테마 계급장
- 출처: Verdict + Sherdog Forum
- 효과: 매주 재방문 명분. 자존심 박제.

### C. 큰 와우 (2-3주 / 2차 고도화) — **로드맵 후크**

#### C-1. **라이브 매치 스레드 자동 생성**
- 이벤트 시작 30분 전 자동 게시 + 실시간 댓글 스트림 + 결과 자동 푸시
- 출처: Reddit Live Fight Thread
- 효과: 같이 보는 경험. 대회 당일 동시접속 폭발.

#### C-2. **선수 가치(인기) 그래프 — 시간축 변동**
- GOAT 투표 변동을 주식 차트처럼 표시
- 출처: Transfermarkt 시장가치 그래프
- 효과: 데이터 시각화 "오 이거 진짜 사이트네"

#### C-3. **AI / 자연어 데이터 검색**
- "맥그리거 KO 승리 몇 번?" → 결과 카드 + 일러스트
- 출처: StatMuse
- 효과: 데모용 한 방. 단 비용/검수 필요.

#### C-4. **스포일러 차단 토글**
- 대회 직후 진입 시 결과 가리기 1탭 토글
- 출처: Reddit 스포일러 정책
- 효과: 늦게 보는 한국 팬 페인포인트 정확히 타격.

#### C-5. **체육관/현역 인증 뱃지**
- 주짓수 퍼플벨트, 현역 선수, 체육관 관장 인증
- 출처: Blind 회사 인증
- 효과: "전문 커뮤니티" 결정타. 단 인증 워크플로우 필요.

### D. 운영자 와우 (관리자 페이지 측면)

#### D-3. **신고 3회 자동 숨김 + 블라인드 큐**
- 1인 운영 가능 수준의 모더레이션

## 5. 한국 시장 차별화 포인트

세 리서치 모두 글로벌 사이트 기반이지만, 한국 진입엔 다음이 결정적:

1. **한국 통용 표기 표준화** — 프라치스, 정찬성, 하빕 (이미 완료)
2. **모바일 첫 화면 = 이번 주 경기 + 1탭 예측** — Sofascore식 캘린더 X, 카드형으로
3. **DC/FMKorea식 베스트 댓글 + 추천 시스템** — 글로벌 사이트엔 없는 감각
4. **판정 논란 자동 투표** — "왜 저걸 졌다고 줌?" 한국 팬 행동 패턴 정확히 타격
5. **격투기 계급장 게이미피케이션** — 화이트벨트→블랙벨트, 한국 커뮤니티 정착도 ↑
6. **스포일러 차단 토글** — 시차로 늦게 보는 한국 팬 페인포인트
7. **선수 한글 별명 표준** (이게 곧 SEO 자산) — "코리안 좀비", "원더보이" 등
