인기투표 랭킹

카테고리별 인물 인기투표와 실시간 TOP10 랭킹을 제공하는 웹서비스 MVP의 구현 내용과 사용법을 안내합니다.

2026.03.14 Next.js 16 + TypeScript Vercel 배포
1

서비스 개요

다양한 카테고리(축구선수, K-POP, 영화 등)에서 인물에게 투표하고, 실시간 TOP10 랭킹을 확인하는 서비스입니다.

🏆

실시간 랭킹

1초 단위 자동 갱신으로 투표 결과가 즉시 반영됩니다. TOP10 바 차트와 카드 순위가 실시간으로 변동합니다.

🗳️

비로그인 투표

회원가입/로그인 없이 누구나 투표에 참여할 수 있습니다. 브라우저 핑거프린트로 중복 투표를 방지합니다.

💬

카테고리 댓글

카테고리별로 자유롭게 댓글을 남길 수 있습니다. 부적절한 댓글은 신고할 수 있습니다.

🌙

다크모드 지원

우상단 토글 버튼으로 라이트/다크 모드를 전환할 수 있습니다. 설정은 브라우저에 저장됩니다.

2

둘러보기 가이드

서비스의 주요 기능을 순서대로 체험할 수 있습니다.

1

메인 페이지 진입

사이트에 접속하면 히어로 섹션(전체 통계)과 TOP3 인기 카테고리가 카드 형태로 표시됩니다. 아래로 스크롤하면 전체 9개 카테고리를 확인할 수 있고, 더보기 버튼으로 추가 로드됩니다.

2

카테고리 상세 페이지

원하는 카테고리를 클릭하면 상세 페이지로 이동합니다. 상단에 TOP10 바 차트가 실시간 그래프로 표시되고, 그 아래에 전체 인물 카드가 순위별로 나열됩니다. 좌상단 LIVE 인디케이터가 실시간 갱신 상태를 보여줍니다.

3

투표하기

각 인물 카드의 "투표하기" 버튼을 클릭하면 투표됩니다. 투표 후 버튼이 보라색 테두리의 "투표 완료"(♥ filled)로 변경됩니다. 같은 버튼을 다시 클릭하면 투표가 취소됩니다 (토글 방식). 다른 인물에게는 각각 투표 가능하지만, 같은 인물에게 중복 투표는 불가합니다.

4

+1 / -1 플로팅 애니메이션

투표/취소 시 버튼 위에서 "+1" 또는 "-1" 텍스트가 위로 떠오르며 사라지는 애니메이션이 재생됩니다. 투표 수 변동은 즉시(낙관적 업데이트) 화면에 반영됩니다.

5

댓글 작성

카테고리 상세 페이지 하단에 댓글 영역이 있습니다. 닉네임(최대 20자)과 내용(최대 300자)을 입력 후 등록 버튼을 누르면 바로 댓글이 표시됩니다. 자신이 작성한 댓글에는 "나" 뱃지가 붙습니다.

6

댓글 신고

다른 사용자의 댓글에 마우스를 올리면 깃발 아이콘이 나타납니다. 클릭하면 신고 사유(스팸, 욕설, 허위 정보, 개인정보 노출)를 선택하는 모달이 열립니다. 같은 댓글에 3회 이상 신고가 접수되면 자동으로 숨김 처리됩니다.

7

다크모드 전환

화면 우상단의 달/해 아이콘 버튼을 클릭하면 다크모드와 라이트모드를 전환할 수 있습니다. 선택한 테마는 브라우저에 저장되어 다음 방문 시에도 유지됩니다.

💡
Tip — 모든 페이지는 모바일 반응형으로 구현되어 있습니다. 브라우저 폭을 줄이거나 모바일 기기에서도 동일하게 체험할 수 있습니다.

4

투표 시스템

핑거프린트 기반 중복방지와 토글 방식의 투표 로직이 구현되어 있습니다.

투표 규칙

✓ 가능 여러 인물에게 각각 1표씩 투표 ✓ 가능 투표 취소 (재클릭)
✕ 불가 같은 인물에게 중복 투표 ℹ 식별 FingerprintJS 브라우저 지문
ℹ️
현재 MVP는 서버 메모리 기반 Mock 저장소를 사용합니다. 서버가 재시작되면 투표/댓글 데이터가 초기화됩니다. 실 서비스 시 Supabase DB로 전환될 예정입니다.

7

관리자 페이지 (/admin)

서비스 운영에 필요한 승인/신고 관리 기능을 갖춘 관리자 콘솔입니다.

📊

대시보드

총 투표수, 승인 대기 카테고리/인물, 신고 접수 건수를 요약 카드로 한눈에 확인합니다. 최근 승인 대기 및 신고 접수 목록도 빠르게 확인 가능합니다.

📋

승인 관리

사용자가 신청한 카테고리와 인물 등록을 데이터 테이블에서 확인하고 승인/거절 버튼으로 처리합니다. 처리 상태가 즉시 뱃지로 반영됩니다.

🚨

신고 관리

신고된 댓글의 내용, 사유, 신고 횟수를 확인하고 숨김/복원/삭제 처리합니다. 이미 자동 숨김된 댓글도 관리할 수 있습니다.

🖥️

SaaS 스타일 레이아웃

좌측 사이드바(탭 네비게이션) + 우측 콘텐츠 영역의 실무 관리 도구 구성입니다. 대기 건수 뱃지로 처리가 필요한 항목을 즉시 파악할 수 있습니다.

1

관리자 콘솔 진입

URL에 /admin을 입력하면 관리자 콘솔로 진입합니다. 좌측 사이드바에서 대시보드 / 승인 관리 / 신고 관리 탭을 전환할 수 있습니다.

2

대시보드 확인

총 누적 투표수, 승인 대기 카테고리(3건), 승인 대기 인물(5건), 신고 접수(2건) 등의 요약 카드를 확인합니다. 하단에는 최근 대기 항목과 신고 내역이 표시됩니다.

3

승인 관리 — 카테고리 및 인물

승인 관리 탭에서 신규 카테고리 등록 신청(예: "2025 올해의 드라마")과 인물 등록 신청(예: "비니시우스 주니오르")을 확인합니다. 각 행의 승인/거절 버튼을 클릭하면 상태가 즉시 변경됩니다.

4

신고 관리 — 댓글 처리

신고 관리 탭에서 신고된 댓글의 내용과 사유(스팸, 욕설 등)를 확인합니다. 숨김 버튼으로 노출을 차단하거나, 삭제 버튼으로 완전히 제거할 수 있습니다. 숨김 처리된 댓글은 복원도 가능합니다.

ℹ️
현재 관리자 페이지는 더미 데이터 기반 시연용입니다. 실 서비스 시 로그인 인증과 DB 연동이 추가될 예정입니다.
8
