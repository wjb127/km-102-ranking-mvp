클라이언트 응답 - 다크모드 / 국적 번역 / 선수 초기화

문의 주신 세 가지 건 안내드립니다. 1. 사이트 기본 테마 기본은 라이트모드로 설정되어 있습니다. 우측 상단 달/해 아이콘 버튼으로 라이트/다크 전환 가능하며, 한 번 다크모드를 선택하시면 같은 브라우저에서는 이후에도 다크로 유지됩니다. 처음 들어가셨을 때 다크로 보이셨다면 이전 접속 때 토글이 눌려 저장된 상태일 가능성이 높습니다. 라이트로 변경하시면 다시 라이트로 고정됩니다. 만약 신규 방문자도 항상 라이트로 시작하도록 저장 자체를 끄는 방향이 더 좋으시면 말씀 주세요. 2. 국적 번역 가능합니다. DB에 이미 국적 한글 컬럼이 마련돼 있고, 현재 선수 이름과 이벤트명 번역에 쓰는 DeepL 배치 스크립트에 국적 항목을 추가해서 일괄 번역 처리해드리겠습니다. 한 번 실행으로 전체 선수 국적이 한글로 채워지고, 이후 신규 선수는 동기화 시점에 자동 번역되도록 연결합니다. 3. 선수 중복/오류 초기화 초기화 가능합니다. 다만 GOAT 투표 데이터가 선수 ID에 연결돼 있어, 진행 방식을 두 가지로 나눠 안내드립니다. a. 중복/오류만 정리 (권장) 같은 선수가 중복으로 들어간 행을 머지하고, 잘못된 데이터만 보정하는 방식입니다. 기존 투표/게시판 데이터는 그대로 보존됩니다. 작업 후 별도 안내 없이도 사이트는 정상 운영 상태 유지됩니다. b. 전체 재동기화 선수 테이블을 비우고 외부 API에서 새로 받아오는 방식입니다. 가장 깨끗하지만 기존 투표 기록이 새 선수 ID와 다시 매칭되어야 하므로, 외부 ID 기준으로 재연결 작업이 필요합니다. 진행 시 30분 내외 점검 시간이 발생할 수 있습니다. 특별한 의도가 없으시면 (a) 방향으로 진행하는 게 안전합니다. 어떤 방향이 좋으실지 회신 주시면 즉시 작업 들어가겠습니다. 추가로, 특정 선수나 항목에서 어떤 오류가 보이셨는지 예시 한두 개만 알려주시면 같은 패턴의 이슈를 일괄 점검하는 데 큰 도움이 됩니다.
